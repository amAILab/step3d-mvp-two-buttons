# STEP_3D MVP — Узнать / Создать

Мобильный MVP для Telegram WebApp/landing-сценария STEP_3D.

## Пользовательский путь

Главный экран начинается с двух действий:

- **Узнать** — быстрый запрос на возможность, цену и срок без обязательства заказа.
- **Создать** — заявка на изготовление с описанием, файлами, материалом, сроком и контактом.

## Разработка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация

Проект публикуется через GitHub Pages workflow:
